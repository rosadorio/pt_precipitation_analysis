O utilizador do dataset PT02 compromete-se a não utilizar os dados para fins comerciais e a não ceder os dados a terceiros sem a autorização do IPMA.

Qualquer publicação ou estudo realizado, utilizando parcialmente ou na totalidade os dados do dataset PT02, deve incluir os seguintes agradecimentos e citação:

Os autores agradecem ao IPMA pelos dados utilizados neste estudo (Dataset de precipitação PT02)
Belo-Pereira, M., E. Dutra, and P. Viterbo, 2011: Evaluation of global precipitation data sets over the Iberian Peninsula, J. Geophys. Res., 116, D20101, 
doi:10.1029/2010JD015481
